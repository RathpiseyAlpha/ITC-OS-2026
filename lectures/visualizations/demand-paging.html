Demand Paging & Virtual Memory — Swap In / Swap Out

A page is brought into RAM only when needed. Pages live on the disk backing store; the valid–invalid bit says whether a page is currently in a frame. Watch how virtual memory can be larger than RAM, and how several programs share the same frames.
