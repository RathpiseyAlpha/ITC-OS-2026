# 📝 Lecture Notes

This folder contains **note-taking templates** for each weekly topic of the Operating Systems course. Each template follows a standardized format to help you develop consistent, effective study notes.

## How to Use

1. **Before class** — Open the template for the upcoming week and skim the key questions.
2. **During class** — Fill in definitions, diagrams, and key takeaways as the lecture progresses.
3. **After class** — Complete the reflection section, add examples, and write your summary.

## Template Structure

Every note file follows this structure:

| Section | Purpose |
|---------|---------|
| **Overview** | One-sentence topic summary you write yourself |
| **Key Concepts** | Core definitions and terminology |
| **Detailed Notes** | In-depth explanations organized by subtopic |
| **Diagrams & Visuals** | Space to sketch or describe diagrams |
| **Key Comparisons** | Tables comparing related concepts |
| **Examples & Code** | Concrete examples or pseudocode |
| **Review Questions** | Self-test questions to check understanding |
| **Connections** | How this topic links to previous/future weeks |
| **Reflection** | What you found challenging, what clicked |
| **Summary** | Your own 3–5 sentence summary of the topic |
